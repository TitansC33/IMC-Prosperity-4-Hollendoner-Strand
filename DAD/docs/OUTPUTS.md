## How Outputs Are Configured

Each script that generates files has been updated with:

```python
from pathlib import Path

# Ensure outputs go to DAD/scripts
OUTPUT_DIR = Path(__file__).parent
OUTPUT_DIR.mkdir(exist_ok=True)

# Then use OUTPUT_DIR when saving:
plt.savefig(OUTPUT_DIR / "chart.png")
with open(OUTPUT_DIR / "output.txt", "w") as f:
    f.write(...)
```

This ensures **all outputs save to DAD/scripts/ regardless of where the script is run from**.
